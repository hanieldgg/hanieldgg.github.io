my work

High School Financial Assistant Web Application
Previous Next

This web based application will serve to help the financial services of a high school to get information from the parents about their children, it will then calculate the costs for each children and add it up to finally be able to print by being imputed in a PDF file.

This application included the system of profiles, so each family would have its own profile where they’d log in using a username and password to register each children separated since this school offered a bunch of extracurricular activities.

Once the form was submitted, the information would go then to a database so the financial administration would be able to make changes if necessary and to retrieve this information when necessary.

This project was all made from scratch using HTML5, PHP, MySQL and Bootstrap.
